 JSR-000107 JCACHE - Compatible Implementations

JSR-000107 JCACHE - Java Temporary Caching API
Compatible Implementations

Once a JSR has posted a Final Release of the Specification, the Maintenance Lead supplies a list of all implementations of the specification that have been certified as compatible, as described in Section 1.4 of the Java Community ProcessSM Program, version 2.9. This list was last updated on 23 September 2014.

Java Temporary Caching API
  • Apache Ignite 1.0
    https://ignite.incubator.apache.org/download.html
    Docs: http://apacheignite.readme.io/v1.0/docs
    apacheignite.readme.io/v1.0/docs/jcache

    Maven: 

    <dependency>
   <groupId>org.apache.ignite</groupId>
   <artifactId>ignite-core</artifactId>
   <version>1.0.2</version>
</dependency>
  • Blazing Cache:
    http://blazingcache.org
    Docs: https://blazingcache.readme.io

    Maven: 

    <dependency>
   <groupId>org.blazingcache</groupId>
   <artifactId>blazingcache-jcache</artifactId>
   <version>1.9.0</version>
</dependency>
  • cache2k:
    https://cache2k.org

    <dependency>

   <groupId>org.cache2k</groupId>
   <artifactId>cache2k-jcache</artifactId>
   <version>0.28-BETA</version>

</dependency>
  • Caffeine
    Download: caffeine-2.2.2.jar
    Docs: caffeine/readme.md

  • Coherence
    http://www.oracle.com/us/products/middleware/cloud-app-foundation/coherence/overview/index.html

  • Ehcache
    https://github.com/ehcache/ehcache-jcache

  • Hazelcast
    http://download.hazelcast.com/downloads/hazelcast-3.3.1.zip
    Docs: http://docs.hazelcast.org/docs/3.3/manual/html-single/hazelcast-documentation.html#hazelcast-jcache-implementation

    Maven: 

    <groupId>com.hazelcast</groupId>
<artifactId>hazelcast</artifactId>
<version>3.3.1</version>
  • Infinispan 7.0.2
    http://infinispan.org/download/
    Docs: http://infinispan.org/docs/7.0.x/index.html
    Using Infinispan as a JSR107 (JCache) provider

  • Redisson:
    http://redisson.org

  • Triava Cache:
    https://github.com/trivago/triava

    <dependency>

   <groupId>com.trivago</groupId>
   <artifactId>triava</artifactId>
   <version>0.9.8</version>

</dependency>
  • WebSphere eXtreme Scale
    https://www.ibm.com/developerworks/downloads/ws/wsdg/
    Version 8.6.1.1
    Java SE provider and Version tested with: IBM J9 VM (build 2.7, JRE 1.7.0 Linux amd64-64 Compressed References 20160630_309914 (JIT enabled, AOT enabled)
