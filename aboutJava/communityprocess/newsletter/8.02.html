 The Java Community Process Program -- August 2002 Newsletter


Java Community ProcessSM — August 2002 Newsletter 

JCP News Bulletin 
August 2002

Here is your August edition of the PMO News Bulletin.  This Bulletin is
sent to JCP Members, Spec Leads and Experts and is scheduled to be a
monthly insert in your email inbox. This edition covers JavaOne Japan,
EC Elections,  the JSR Naming Program and the JCP 2.5 and JSPA JSRs.


** Recent News **
----------------------
Find out about more about the latest happenings in the Java Community
Process (sm) (JCP) program, by visiting  the "What's New" area of the
JCP web site:
http://www.jcp.org/whatsnew/jcp/index.jsp.

** JavaOne(sm) Japan 2002 **
 -----------------------------------
JavaOne Japan will be held in Yokohama, Japan, September 25-27, 2002.
Meet and network with Java (tm) technology developers from around the
world to help take your code to the next level.

The JCP will host an evening reception for JCP Members, immediately
following the Java Licensee event.
Stay tuned for more details.

For 2002 Conference information, go to: http://java.sun.com/javaone/


** JCP EC Elections: October 2002 - November 2002 **
 ----------------------------------------------------------
Preparations for the Executive Committee (EC) Elections are underway,
with the ratification ballot opening on October 1.

The JSPA contacts for each JCP member company are eligible to vote and
will be sent the voting materials. To confirm that we have the correct
JSPA contact within your company please send e-mail to
participant-access@jcp.org.

Following the ratifications, nominations for the open elections can be
submitted. If you or your company is interested in joining one of the
ECs this is a good time to prepare your nomination. The nomination
template will become available from the web site on October 17.


**Want to use the Java trademark in your JSR Name?**
------------------------------------------------------------
Sun announces Early Access to JSR Naming Program.   This program allows
the use of the Java  name in your JSR and the resulting specification,
by meeting specific quality standards. This new program applies only to
JSRs submitted to the JCP PMO with specification titles that use the
Java brand name in the form "Java(TM) " (e.g., "Java(TM)
Braille API Specification"). Such JSRs must meet the quality criteria
defined by this program. Existing Java technologies and currently
accepted JSRs using the Java brand name in this manner are not required
to follow this program.

The final version of the program will be available after Early Access
has been completed.  Please send any questions to jsrnaming@sun.com.

For full details see: http://java.sun.com/programs/jsr-naming/


** Noteworthy JSRs **
-------------------------
This section of the News Bulletin calls out JSRs that have news to share

with the Java Community. If your JSR has something newsworthy to report
then send an alert ("Hey PMO!") to pmo@jcp.org.

-JSR 99, Java Specification Participation Agreement (JSPA)
This JSR is the major revision of the JSPA.  It is currently is Public
Review; this review closes August 7.
http://jcp.org/jsr/detail/99.jsp

-JSR 171, Java Community Process (JCP) Program, version 2.5
This JSR aims to define a new version of the Java Community Process. Its

overriding goal is to ensure that the Process document is in sync with
the new version of the JSPA under development by JSR 99.  It is
currently in Community Review; this review closes August 12.
http://jcp.org/jsr/detail/171.jsp

When submitting a proposal for a new JSR please use the JSR Proposal
template found at: http://www.jcp.org/jsr/proposal/index.jsp
